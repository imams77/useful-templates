# Templates

This folder contains markdown templates you can copy and fill in for common artifacts.

Files:

- `agent-file-template.md` — starter for AI agent configuration or instructions.
- `checklist-template.md` — checklist skeleton for reviews and releases.
- `plan-template.md` — planning template for scoped work.
- `spec-template.md` — spec template for feature or design documents.
- `tasks-template.md` — task breakdown template for execution.

How to use:

1. Copy the template into your project (e.g., `.specify/templates/` or a docs folder).
2. Fill in the sections and share with your team for feedback.
3. Use these templates with AI prompts to accelerate generation and review.
